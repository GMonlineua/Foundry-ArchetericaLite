# CHANGELOG

## Version 0.2 Alpha
Оновлені стилі аркушів персонажів і ніпів, виправлена проблема з локалізацією листа персонажа.

## Version 0.3 Alpha
Оновлені стилі елементів, списків елементів та добавлений фон світу за замовчуванням.
